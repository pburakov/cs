# Simple Regex Matcher 

Implement a simple regular expression matcher supporting following expressions:
 - a dot `.` matches a single character
 - a star `*` matches zero or arbitrary repeated characters

Examples:
```
R="a*c", S="abc" -> True
R="a.c", S="abbc" -> False
```

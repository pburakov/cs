In a two-dimensional matrix or integers, find the length of a path of increasing numbers.
 Ony vertical or horizontal moves are allowed (no diagonals).

Example:
```
Input: 
   [[3, 6, 7, 5],
    [2, 8, 9, 1],
    [4, 8,10, 0],
    [6, 7, 3, 9]]
Output: 6
(path: 2->3->6->7->9->10)
```

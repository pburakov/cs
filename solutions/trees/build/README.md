# Build a Binary Search Tree

Implement an algorithm to build a BST from a sorted array of comparable objects (integers).

Example:
```
Input: [1, 2, 3, 4, 5, 6]
Output:
      3
    /   \
   1     5
    \   / \
     2 4   6
```

Return the root of a tree.
